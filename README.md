## 🎯 What is Typing Arena?

**Typing Arena** is a clean, minimalist, real‑time typing speed tester that runs entirely in your browser. No installations, no distractions – just you and the keyboard.

- 📊 **Live WPM calculation** – updates as you type  
- ⏱️ **Accurate timer** – starts only when you begin  
- 🎲 **Random sentences** – 25+ medium‑length quotes to keep it fresh  
- 🎨 **Modern gradient UI** – easy on the eyes, focused on the task  

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Real‑time WPM** | See your words‑per‑minute update after every keystroke |
| **Automatic timer** | Starts the moment you type the first character |
| **Accuracy check** | The test only completes when the entire sentence matches |
| **Random sentences** | A new sentence every reset – never type the same thing twice |
| **Responsive design** | Works on desktop, tablet, and mobile devices |
| **Zero dependencies** | Pure HTML, CSS, and JavaScript – no frameworks, no npm |
